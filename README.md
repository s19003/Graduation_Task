## 卒研プロトタイプ(Graduation_Task)

### 使い方

```
git clone https://github.com/s19003/Graduation_Task.git
```

```
npm install
```

```
node app.js
```

### 構成

- サーバ<br>

  - app.js
  - twitter.js

- フロント<br>
  メインとなる操作用画面

  - main.html
  - main.css
  - main.js

  OBS に映すコメント画面

  - screen.html
  - screen.css
  - niconico.css <- ニコニコ風 CSS
  - youtube.css <- Youtube 風 CSS
  - screen.js

  共通の値

  - config.js
  - img ディレクトリ
